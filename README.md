1. Moodle2.css
   - di halaman Site Administration -> Appearence -> Themes -> Snap
   - Masukkan code ke kolom Custom CSS di paling bawah
   - Save Changes
2. aboutus.html
   - di halaman Homepage, klik pengaturan di pojok kanan atas
   - di bawah Front page settings, klik Turn editing on
   - scroll ke tombol Edit summary
   - pilih mode HTML di text editor
   - copy code aboutus.html 
   - Save Changes
3. explore.html
   - dengan mode editing on, scroll ke bawah dan klik add an activity or resource
   - pilih label
   - klik mode HTML di text editor yang ada
   - copy code explore.html
   - Save Changes
4. contactbtn.html
   - di halaman Site Administration -> Appearence -> Additional HTML
   - Scroll kebawah
   - copy code contactbtn.html ke kolom Before BODY is closed
   - Save Changes
